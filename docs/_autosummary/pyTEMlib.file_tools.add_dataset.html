pyTEMlib.file_tools.add_dataset¶

pyTEMlib.file_tools.add_dataset(dataset, h5_group=None)[source]¶

Write data to hdf5 file

Parameters

  • dataset (sidpy.Dataset) – data to write to file

  • h5_group (None, sidpy.Dataset, h5py.Group, h5py.Datset, h5py.File) – identifier to which group the data are added (if None the dataset must have a valid h5_dataset)

Returns

log_group – reference the dataset has been written to. (is also stored in h5_dataset attribute of sipy.Dataset)

Return type

h5py.Dataset
